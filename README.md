# คำอธิบายฟังก์ชัน

# `myfunc(a, b)`

- a: ตัวอักษรที่ต้องการซ้ำ
- b: จำนวนครั้งที่ต้องการซ้ำ
- Return: สตริงที่ประกอบด้วยตัวอักษร a ซ้ำ b ครั้ง

ตัวอย่างการเรียกใช้:

```python

myfunc("( •_•)", 5)

```
